Aqui está um **Script de Prompt** altamente detalhado e estruturado para você copiar e colar no Cursor (ou qualquer AI de codificação).

Ele foi desenhado para garantir que o Cursor entenda não apenas o código, mas a **filosofia de design** (Steve Jobs/Apple), a **paleta de cores específica** e a **stack técnica** (Expo/Reanimated/Gemini) que definimos.

### **Como usar:**

1. Abra o **Cursor**.  
2. Crie ou abra o arquivo onde você quer a tela (ex: app/index.tsx).  
3. Abra o chat do Cursor (Cmd+K ou Ctrl+K no código, ou o painel de Chat lateral).  
4. **Cole o script acima.**  
5. O Cursor irá gerar o código React Native otimizado respeitando exatamente as cores hexadecimais e a lógica que criamos juntos.
